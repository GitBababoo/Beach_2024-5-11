# 🤖 BOT กดสินค้าอัตโนมัติ

BOT สำหรับกดสินค้าอัตโนมัติที่สามารถทำงานกับเว็บไซต์ขายสินค้าต่างๆ ได้อย่างรวดเร็วและมีประสิทธิภาพ

## ✨ ฟีเจอร์หลัก

- **🔐 ล็อกอินอัตโนมัติ** - เข้าสู่ระบบโดยอัตโนมัติ
- **🔍 ค้นหาสินค้าอัตโนมัติ** - ค้นหาสินค้าที่ต้องการได้อย่างรวดเร็ว
- **🛒 เพิ่มลงตะกร้า** - เพิ่มสินค้าลงตะกร้าอัตโนมัติ
- **💳 ชำระเงิน** - ชำระเงินอัตโนมัติ
- **👀 ตรวจสอบราคา** - ตรวจสอบราคาสินค้าแบบเรียลไทม์
- **🌐 รองรับหลายเว็บไซต์** - สามารถปรับแต่งให้ทำงานกับเว็บไซต์ต่างๆ ได้

## 🚀 การติดตั้ง

### 1. ติดตั้ง Dependencies

```bash
pip install -r requirements.txt
```

### 2. เริ่มต้นใช้งาน

```bash
python app.py
```

### 3. เปิดเว็บอินเตอร์เฟซ

เปิดเว็บเบราว์เซอร์และไปที่: `http://localhost:5000`

## 📋 การใช้งาน

### วิธีการใช้งานพื้นฐาน

1. **กรอกข้อมูลเว็บไซต์**
   - ใส่ URL ของเว็บไซต์ขายสินค้า
   - ใส่ชื่อผู้ใช้และรหัสผ่าน

2. **เลือกสินค้าที่ต้องการ**
   - ใส่ชื่อสินค้าที่ต้องการซื้อ
   - ระบุจำนวนที่ต้องการ

3. **เริ่มการทำงาน**
   - กดปุ่ม "เริ่มกดสินค้า"
   - BOT จะทำงานอัตโนมัติตามขั้นตอน

### ตรวจสอบราคา

1. กรอกข้อมูลเหมือนขั้นตอนพื้นฐาน
2. กดปุ่ม "ตรวจสอบราคา"
3. ใส่ราคาเป้าหมาย
4. BOT จะตรวจสอบราคาและซื้ออัตโนมัติเมื่อถึงราคาเป้าหมาย

### สำหรับเว็บไซต์เฉพาะ

## 🔧 การแก้ไขปัญหา

### เกิดข้อผิดพลาดในการล็อกอิน

- ตรวจสอบชื่อผู้ใช้และรหัสผ่าน
- ตรวจสอบว่าเว็บไซต์รองรับการล็อกอินแบบอัตโนมัติหรือไม่
- อาจต้องเพิ่มการจัดการ Captcha

### ไม่พบสินค้าที่ต้องการ

- ตรวจสอบชื่อสินค้าที่พิมพ์
- อาจต้องปรับแต่งฟังก์ชันการค้นหา
- ลองใช้คำค้นที่แตกต่างกัน

### เกิดข้อผิดพลาดในการชำระเงิน

- ตรวจสอบข้อมูลการชำระเงิน
- อาจต้องเพิ่มการจัดการกับระบบป้องกัน BOT
- ตรวจสอบยอดเงินคงเหลือ

## ⚠️ คำเตือน

- **ใช้อย่างมีความรับผิดชอบ** - อย่าใช้เพื่อการเก็งกำไรหรือเอาเปรียบผู้อื่น
- **ตรวจสอบกฎหมาย** - ตรวจสอบว่าการใช้ BOT ไม่ผิดกฎหมายของเว็บไซต์
- **ความเสี่ยง** - การใช้ BOT อาจถูกแบนจากเว็บไซต์
- **ข้อมูลส่วนตัว** - อย่าแชร์ข้อมูลล็อกอินกับผู้อื่น

## 📝 ตัวอย่างการใช้งาน

```python
from bot_structure import ProductBot

# สร้าง BOT
bot = ProductBot(
    base_url="https://example-shop.com",
    username="your_username",
    password="your_password"
)

# ซื้อสินค้า
bot.auto_buy("iPhone 15", quantity=1)

# ตรวจสอบราคา
bot.monitor_product("iPhone 15", 30000)

bot.close()
```
